      • Changed color to light red, to stand out better.

      • Changed the default avatar to a letter avatar, that will show initials.

      • If the LCEC IT Administration role is selected the user will be able to manually change prices add new prices or remove prices. No other Role can perform this action.

      • Added functionality to Inspection review page to allow Admins to remove voucher's from an Inspection.
    • Manually trigger sync for single Word Order - Admin feature request
      • Added a page for Admins only to try to pull in a WO that is not able to pull in from the normal sync job that runs every hour.

      • Added the ability for vendors to modify a voucher only if it has not been inspected or invoiced.

    Release v1.0.0 - HC

    September 27, 2018

    Features Added:

    • Dashboard
      • Shows basic statistics on areas released in this version
    • Staking Area
      • Search
        • Can search for work orders
      • As-Built Tab
        • LCEC can present a work order’s design to a vendor
        • Vendor can update the actual As-Built quantities and submit for inspection
      • Inspection Tab
        • LCEC can inspect assemble units
        • Vendor can review the results of those inspections
      • Invoicing Tab
        • Vendor can submit an invoice for payment to LCEC
        • LCEC can review invoice for approval or rejection
      • Summary Tab
        • Can see an aggregated view of all assemble units on the work order
    • Streetlights Area
      • Search
        • Can search for work orders
      • Inspection Tab
        • LCEC reviews vendor submitted paper forms and can mark the work order ready for invoicing
      • Invoicing Tab
        • Vendor can submit an invoice for payment to LCEC
        • LCEC can review invoice for approval or rejection
    • Inspection Area
      • Search
        • Can look for inspections in staking and streetlights based upon the user’s criteria
        • Inspectors can use as centralized list of invoices to approve
    • Admin Area
      • Used to create and manage users and their roles

    Known Issues:

    • Service orders for streetlights which require the entry of feet of cable is not supported in this release
    • No work order with less than 9 characters will be found in the VendorPortal
    • WorkOrder starting with the number “3” will not be found in VendorPortal
    • The Following Service Order Types will not be found in VendorPortal
      • APT, CNMNR, CNTB, DCLAIM, DMTNC, ER, HP, PSC, OSP, TPOH
